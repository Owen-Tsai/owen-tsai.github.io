在过去的几个月，Harness Engineering 已经代替 Context Engineering 和 Prompt Engineering，成为了 AI 领域最热门的词汇。在[OpenAI 发表的文章](https://openai.com/zh-Hans-CN/index/harness-engineering/)中，详细记载了如何通过 harness 使 Codex 交付了一款完全不包含人工编码的产品。这篇文章背后的工程实践具有革命性意义，让人们（再次）看到了 agent 在产品开发和交付上潜在的能力。
