---
title: FusionX
idx: 2
---

FusionX 是一款通过可视化界面和配置式方式进行业务应用开发的低代码开发平台。

FusionX 依托企业私有低代码协议，使用 Schema 描述构成应用程序的组件、页面、事件、数据模型等元信息。结合代码生成和运行时渲染引擎，实现降低开发门槛、快速交付业务的目标。

在 FusionX 中开发应用，开发者需要首先配置数据源。选取与业务关联的数据表，FusionX 引擎将根据数据表的字段元信息（名称、备注、类型等），生成初始 Schema 和服务端模块。开发者在初始 Schema 的基础上进行开发，例如添加更多组件、配置事件或生命周期、定义远程 API 和设置字段的校验规则等。

在灵活性方面，FusionX 暴露设计器和渲染器组件，以及丰富的 API 与生命周期钩子。无论是完全通过低代码设计器进行开发，还是通过调用渲染器的 API 实现 pro code 与 low code 的混合开发，FusionX 都能胜任。
